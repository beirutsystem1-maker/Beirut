# 🚀 Antigravity: Sistema Dinámico de Gestión de Cobranzas

Sistema moderno y automatizado para el control de ventas a crédito, facturación y recuperación de cartera, diseñado para optimizar el trabajo en áreas de ventas y administración.

## ✨ Características Principales
* **Visualización por Estados**: Filtros interactivos para facturas "En Mora", "Pendientes" y "Al Día".
* **Abono en Cascada (FIFO)**: Los pagos se acreditan automáticamente a las deudas más antiguas.
* **Historial Mensual Inteligente**: Organización automática de facturas pagadas en contenedores por mes de emisión.
* **Cobranza Digital**: Generación de fichas de pago listas para compartir por WhatsApp.
* **Ejecución Silenciosa**: El sistema corre en segundo plano sin ventanas de terminal visibles.

## 🛠️ Instalación en una Nueva PC
Este proyecto está diseñado para ser portatil y de fácil despliegue.

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/antigravity-cobranzas.git
   ```

2. **Ejecutar el Instalador**:
   - Navega a la carpeta del proyecto clonado.
   - Haz doble clic en el archivo `install.bat`.
   - Este script preparará el entorno automáticamente y creará un acceso directo en tu Escritorio.

3. **Inicia el Sistema**:
   - Haz doble clic en el acceso directo del escritorio ("Abrir Beirut" o "Antigravity").
   - ¡Listo! El entorno se ejecuta silenciosamente y lanzará tu navegador al panel principal.
